## Tech Debt

**Phase 01**
- `cache.js` schemaFingerprint is a sorted table-name list. Column-level schema changes (add/remove column) without table changes won't invalidate cached queries. Acceptable for demo scale.

**Phase 02**
- `apiFetch` helper exported from `client/src/api.js` but never imported anywhere — dead code.
- `RateLimitBanner` receives only `remaining` prop. The `reset` timestamp is available in `DbExplorer.rateLimitInfo` but not forwarded. The nav bar `RateLimitBadge` does show time-until-reset; the chat banner does not.

## DEPL-06 — Supabase Demo DB (Open Action)

**Status:** Requires manual Railway dashboard action — not verifiable from code.

To satisfy DEPL-06: Set in Railway environment variables:
- `DEMO_DB_HOST`
- `DEMO_DB_USER`
- `DEMO_DB_PASSWORD`
- `DEMO_DB_NAME`

The `POST /db/connect-demo` route (`postgresController.connectDemo`) reads these env vars. If not set, demo connect will fail silently or with an internal error.
